# Research on Morphosemantics

This repository contains the supplementary materials and appendices for Yağmur Öztürk's PhD thesis on Modelling Turkish Nominal Derivation: From Linguistic Formalisation to Morphosemantic Knowledge Representation.

## Contents

- **Annotated Corpora**: Evaluation datasets for Semantürk tagsets
- **Annotation Guidelines**: Detailed annotation manual used for corpus annotation
