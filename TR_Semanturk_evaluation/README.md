# Semantürk Evaluation

This folder contains resources and data related to the evaluation of the Semantürk corpus.

## Folder Contents

### Annotated Corpus
- **File**: `corpus_tr_semanturk_annotation.xlsx`
- **Description**: Complete corpus with three completed annotations
- **Usage**: Inter-annotator analysis and annotation consistency evaluation

### Annotation Guide
- **File**: Annotation guide `Part1_Guide_and_Methodology.pdf`
- **Description**: Documentation of annotation guidelines and methodology used for the Semantürk corpus
- **Usage**: Reference for understanding the applied annotation criteria

### Semantürk Hierarchy
- **File**: `hierarchy_of_the_semanturk_ontology.png`
- **Description**: Visual representation of the Semantürk hierarchical structure
- **Usage**: Understanding the taxonomic organisation and relationships between semantic categories

## About Semantürk

Semantürk is a semantic resource that organises concepts according to a hierarchical structure for the annotation of Turkish noun-to-noun derivational morphemes. This evaluation folder enables analysis of the quality and consistency of annotations performed on the corpus.

## Usage

These resources are intended to:
- Evaluate annotation quality
- Calculate inter-annotator agreement
- Validate the annotation methodology
- Document the evaluation process

# Semantürk Evaluation

## À propos de Semantürk

Semantürk est une ressource sémantique qui organise les concepts selon une structure hiérarchique pour l'annotation de morphèmes dérivationnels turcs de noms à noms. Ce dossier d'évaluation permet d'analyser la qualité et la cohérence des annotations réalisées sur le corpus.
